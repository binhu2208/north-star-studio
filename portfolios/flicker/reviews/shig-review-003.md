## What We Built

### Core Loop
- Torch-based roguelike with real-time resource pressure
- 3-level descent with escalating difficulty
- Upgrade system between levels (3 choices)
- Combat system (SPACE to attack, 10% fuel cost)

### Technical
- Single HTML file, ~26KB
- Zero dependencies
- Runs in any modern browser
- FPS-independent mechanics

### Progression
- Level 1: Tutorial feel
- Level 2: 30% harder
- Level 3: Maximum tension
- Victory screen with stats
