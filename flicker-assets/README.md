# Flicker Art Assets

Pixel art assets for Flicker Godot edition.

## Asset Structure

```
exported/
├── sprites/
│   ├── player/       # Player character sprites
│   ├── enemy/        # Enemy (Stalker) sprites
│   ├── environment/  # Tiles and world objects
│   ├── ui/           # User interface elements
│   └── vfx/          # Visual effects
source/
└── aseprite/         # Source .ase files
```

## Style Guide

- **Resolution:** 32x32 base tiles, scaled as needed
- **Style:** Pixel art, limited palette
- **Atmosphere:** Dark dungeon, high contrast
- **Colors:** Deep blues/blacks for shadows, warm oranges/yellows for torch
